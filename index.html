My Coding Projects

ePasaBuyMo

ePasaBuyMo is a full-stack web application that caters to the specific needs of “pasa-buy” entrepreneurs. The user interface is built with ReactJS and Tailwind CSS. Documents are stored in the cloud database of MongoDB atlas. Datas are fetched using a popular NodeJS framework, ExpressJS.

  • ReactJS
  • Tailwind CSS
  • ExpressJS
  • MongoDB

Facebook Front-end Clone Code-along

A facebook front-end clone app built on ReactJS ready to take in functionality where you are able to create posts, authenticate the user, view user profiles, follow and unfollow friends and so much more. It's a very exciting project that can evolve to a full-stack clone very soon.

  • Javascript
  • ReactJS
  • Material UI
  • CSS for styling

Quiz Game on ReactJS

A "Who wants to be a millionaire" trivia game built on ReactJS. It contains some cool functionalities like being able to set a username before the start of the game. It also exhibits the useSound hook of ReactJS. This project is ready to be connected to an API to make the trivia questions dynamic.

  • Javascript
  • ReactJS
  • CSS for styling
  • useSound for Effects
